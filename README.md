# History-of-Video-Games
Open Educational Resource (OER) about the history of video games.
It can be accessed via the [web version](https://niema.net/History-of-Video-Games) ([ZIP file](https://github.com/niemasd/History-of-Video-Games/releases/latest/download/History-of-Video-Games.zip) for offline viewing) or the [PDF](https://github.com/niemasd/History-of-Video-Games/releases/latest/download/History-of-Video-Games.pdf).

# Compilation
You can run this command to compile the resource yourself:

```bash
rm -rf ~/Desktop/out/ && ./scripts/build.py -o ~/Desktop/out
```
