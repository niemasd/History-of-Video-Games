This directory contains all companies. The following attributes are required:

* **`name`** — The name of the company as a string
* **`name_safe`** — A safe version of the company name (e.g. for filenames) as a string
* **`date_start`** — The start (i.e., founding) date of this company as a string in the format `YYYY-MM-DD`

The following attributes are optional:

* **`location_start`** — The initial (i.e., founding) location of this company as a string
* **`founders`** — A list containing the names of the founders of the company as strings
* **`date_end`** — The end (i.e., closing) date of this company as a string in the format `YYYY-MM-DD`
