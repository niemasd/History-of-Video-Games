This directory contains all people. The following attributes are required:

* **`name`** — The English name of the person as a string
* **`date_birth`** — The birth date of the person as a string in the format `YYYY-MM-DD`

The following attributes are optional:

* **`name_orig`** — The original name of the person (e.g. in its original language) as a string
* **`image`** — The path to an image of the person as a string (place images in [`data/images`](../images))
* **`location_birth`** — The birth location of the person as a string
* **`date_death`** — The death date of the person as a string in the format `YYYY-MM-DD`
* **`location_death`** — The death location of the person as a string
